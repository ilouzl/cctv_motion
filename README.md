## Installation
* Make sure you have python3 installed: 
    ```
    sudo apt update && sudo apt install python3 python3-pip
    ```  
* Install package dependancies:
    ```
    sudo apt-get install python3-opencv
    sudo pip3 install -r requirements
    ```
